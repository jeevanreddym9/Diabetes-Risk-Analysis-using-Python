# 🩺 Diabetes Risk Analysis using Python

## 📌 Project Overview
This project analyzes a healthcare dataset to identify patterns and risk factors associated with diabetes. The analysis includes data cleaning, exploratory data analysis (EDA), and custom risk classification based on glucose levels.

---

## 🎯 Objectives
- Perform data preprocessing and cleaning
- Analyze health indicators such as Glucose, BMI, Age, and Blood Pressure
- Classify patients into Risk Levels (High, Medium, Low)
- Compare Risk Level with actual diabetes outcome

---

## 🛠 Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

---

## 📊 Risk Classification Logic

- 🔴 High Risk → Glucose > 140  
- 🟡 Medium Risk → 111–140  
- 🟢 Low Risk → ≤ 110  

---

## 📈 Key Insights
- Higher glucose levels strongly correlate with diabetes.
- High-risk patients show higher probability of being diabetic.
- BMI and Age contribute significantly to diabetes risk.

---

## 📂 Dataset Features
- Pregnancies
- Glucose
- Blood Pressure
- BMI
- Age
- Outcome (0 = Normal, 1 = Diabetic)
